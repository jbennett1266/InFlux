# Project Mission: InFlux

## 1. Executive Summary
- **Goal:** Build an E2E Encryption messaging app
- **Target User:** Semi security concious individuals who want anonymity when texting one another

## 2. Core Features (The "Must-Haves")
- [ ] Feature 1: A frontend, backend, and api layer that supports sending short/small high frequency messages
- [ ] Feature 2: All traffic is encrypted while in transit and never stored in plaintext except on the end device
- [ ] Feature 3: This is not a short range (bluetooth) type application where users must be in close proximity, this is a service that anyone can sign up for with an account
